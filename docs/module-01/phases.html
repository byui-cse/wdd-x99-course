Phases

This project is broken into 4 phases:

  • Phase 1: Planning the design and estimating cost
  • Phase 2: Creating a basic functional web application
  • Phase 3: Decoupling the application components
  • Phase 4: Implementing high availability and scalability

Phase 1

Mary Cordova, CTO

For phase 1, create an architectural diagram to illustrate what you plan to build.

Make sure you include each service you plan on using as well as how you plan on separating networks to keep things secure.

Christina Stone, CFO

For phase 1, I'll need an estimate of the yearly costs associated with the new system.

Phase 2

Mary Cordova, CTO

For phase 2, I'll want to see the current monolithic system working on AWS. After that we can look at how to split things up.

Phase 3

Mary Cordova, CTO

For phase 3, Your deliverable will be showing the system running on different services. The database must to its own service.
